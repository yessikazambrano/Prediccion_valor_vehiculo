# Prediccion_valor_vehiculo
Desarrollé un modelo de regresión para estimar el valor de mercado de coches usados, como parte de una solución para la app de Rusty Bargain, empresa dedicada a la venta de vehículos de segunda mano.El objetivo era ofrecer a los usuarios una estimación del precio de su coche, basada en datos históricos, especificaciones  técnicas y versión.
